# HR Analytics Dashboard

A complete end-to-end **HR Analytics** project built using **Python, MySQL, and Power BI**. The project focuses on analyzing employee data to identify workforce trends, attrition patterns, salary distribution, and key HR metrics through interactive dashboards.

---

## 📌 Project Overview

The goal of this project is to analyze HR employee data and provide actionable business insights for decision-making.

The project includes:

- Data Cleaning using Python
- Exploratory Data Analysis (EDA)
- SQL Data Analysis
- Interactive Power BI Dashboard
- Business Insights & KPI Reporting

# 📌 Key Business Insights

- Research & Development department has the highest number of employees.
- Sales department has the second largest workforce.
- Employee attrition is significantly higher among employees working overtime.
- Average monthly income varies considerably across job roles.
- Employees with lower job satisfaction exhibit higher attrition rates.
- Male employees represent a larger share of the workforce than female employees.
